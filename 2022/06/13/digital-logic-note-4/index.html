数字逻辑笔记 4 - 时序逻辑电路

这一部分对应教材的 Chapter 7~9

一、时序逻辑电路介绍

  时序逻辑电路的一个特点就是，它的输出不仅与输入变量有关，也与前一时刻输入形成的状态有关。
  下面给出几个定义：

  • EEE 是激励 (Excitation) 变量
  • SSS 是状态 (Status) 变量

  有两种基本模型：

  • Moore 模型

    可见，它的输出(O)(O)(O)只和当前状态(St)(S_t)(St​)有关。它的表达式为

E=f(I,St)St+1=f(St,E)O=g(St)E=f(I,S_t)\\ S_{t+1}=f(S_t,E)\\ O=g(S_t) E=f(I,St​)St+1​=f(St​,E)O=g(St​)

  • Mealy 模型

    可见，它与 Moore 模型的区别是，它的输出不仅与当前状态有关，还与输入变量(I)(I)(I)有关。它的表达式为

E=f(I,St)St+1=f(St,E)O=g(I,St)E=f(I,S_t)\\ S_{t+1}=f(S_t,E)\\ O=g(I,S_t) E=f(I,St​)St+1​=f(St​,E)O=g(I,St​)

二、锁存器和触发器 (Latches/Filp-Flops)

  这东西在时序逻辑电路里面的地位和在组合逻辑电路里面的地位相当，是必须要掌握的。一般来说，触发器有四种：SR、D、JK、和 T 型触发器。触发的类型有三种，这里着重介绍的是边沿触发（就是在时钟 (CLK) 信号的上升沿或下降沿触发）。如下图，左边是上升沿触发，右边是下降沿触发。

定义：当触发器的 QQQ 端为高电平时为 SET\rm SETSET 状态，反之为 RESET\rm RESETRESET 状态。

1. SR 锁存器

这是最基础的一种锁存器，其余三种都是由这种衍生而来。这种锁存器有两种实现。
S’R’ 锁存器 (使用 NAND 门实现)

  这是低电平有效 (Active-LOW) 的锁存器，意思是当输入端为低电平时会引起结果的变化。
  经过一些模拟发现，假设现在锁存器是 RESET\rm RESETRESET 状态，在 Sˉ\bar SSˉ 端施加短暂的低电平信号后，锁存器会变为 SET\rm SETSET 状态。相对地，在 SET\rm SETSET 状态下给 Rˉ\bar RRˉ 端施加短暂的低电平信号会使锁存器变为 RESET\rm RESETRESET 状态。而当两端都施加高电平信号时，输出不变 (HOLD)(\rm HOLD)(HOLD)；两端都施加低电平是非法操作，会导致输出结果不确定。以下是真值表

SR 锁存器 (使用 NOR 门实现)

  相反地，这是高电平有效 (Active-HIGH) 的锁存器。在 SET\rm SETSET 状态下，给 RRR 端施加高电平信号会使锁存器变为 RESET\rm RESETRESET 状态，其他不必多说。以下是真值表

加上门控信号的 SR 锁存器

在 SR 锁存器前面加上一段电路，其中的 CCC 端 (书上为 ENENEN 端) 为低电平时，改变锁存器的 SSS 或 RRR 端不会造成影响。

2. D 锁存器

这是最简单的一种锁存器(个人意见)。SR 锁存器存在非法输入，它会导致输出结果不确定，而 D 锁存器就不存在这个问题。

真值表也很简单：

DDD QnQ^nQn Qn+1Q^{n+1}Qn+1 功能
0 0 0 Reset
0 1 0 Reset
1 0 1 Set
1 1 1 Set

特征方程为：

Qn+1=DQ^{n+1}=D Qn+1=D

它与 SR 锁存器类似，但是它允许 JJJ 和 KKK 端有相同的输入，此时 Qn+1Q^{n+1}Qn+1 会发生反转 (TOGGLE)(\rm TOGGLE)(TOGGLE)。以下为真值表

0 0 1 1 Hold

0 1 1 0 Reset
1 0 0 1 Set
1 0 1 1 Set
1 1 0 1 Toggle
1 1 1 0 Toggle

Set 和 Reset 部分和 SR 锁存器是相反的，其余的都是一致的。
它的特征方程为

Qn+1=JQn‾+K‾QnQ^{n+1}=J\overline {Q^n}+\overline KQ^n Qn+1=JQn​+KQn

T 锁存器与 JK 锁存器类似（其实就是把 JJJ 和 KKK 端连起来)，就是少了一个输入。它在 TTT 端为低电平时输出保持不变，在为高电平时将输入反转。以下为真值表

三、触发器 (Flip-Flops)

每一种锁存器都有对应的触发器。触发器与锁存器的区别是，触发器状态的改变时机取决于它接收到的时钟信号。就是这样。

四、状态机 (State Machine) 和状态图 (State Diagram)

一个状态机有以下要素：

  • 输入变量：源于时序机（状态机）外部的所有变量皆被认为是输入变量；
  • 输出变量：离开时序机（状态机）的所有变量皆被认为是输出变量；
  • 状态变量：存储元件（触发器）的输出定义了时序机(状态机)的状态。即触发器的输出就是状态变量。
  • 激励变量：是指对存储元件（触发器）的输入。该激励变量刺激存储元件，使其状态发生改变。
  • 状态：状态变量的确定组合对应一种状态。如果一个状态机有 2n2^n2n 个状态，那么对应了 nnn 个状态变量。

状态图主要由节点和它们间的连线表示。对于两种基本模型，它们的表示方式有些差异。

  • 对于 Mealy 模型，节点表示状态，在连线上标注输入/输出。
  • 对于 Moore 模型，节点表示状态/输出，在连线上标注输入。

五、时序组合逻辑电路的设计和分析

1. 设计

  • 根据需求，得出状态，画出状态图。
  • 根据状态图，得到状态表 (State Table)，里面包含了从当前状态到下一状态的条件和输出等。
  • 把状态用状态变量的组合表示，得到状态转移表 (Transition Table)。
  • 选择需要使用的触发器类型，得到对应触发器的激励表 (Excitation Table)。
  • 把激励表对应到状态转移表中，然后用卡诺图化简各变量，再画图实现即可。

2.分析

  • 先找出系统中的输入、状态和输出变量
  • 根据所使用的触发器类型写出特征方程
  • 根据所给图示写出激励方程
  • 根据特征方程和激励方程，得到转移方程
  • 构造出状态转移表
  • 画出状态图
